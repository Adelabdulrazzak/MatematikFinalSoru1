Proje  Açıklama:
​Bu proje matematiksel bir problem olan "İkinci Dereceden Denklemlerin Köklerini Bulma" işlemini C++ dili kullanarak otomatize etmektedir Programın çalışma mantığı şu şekildedir:
​Veri Girişi: Kullanıcıdan denklemin temel katsayıları olan a, b ve c değerleri alınır.
​Diskriminant Hesabı: Denklemin yapısını belirleyen \Delta = b^2 - 4ac formülü uygulanır.
​Mantıksal Denetim: >    * Eğer \Delta > 0 ise program iki farklı reel kök olduğunu hesaplar.
​Eğer \Delta = 0 ise, çakışık (tek) bir kök olduğunu belirtir.
​Eğer \Delta < 0 ise, reel kök olmadığını kullanıcıya bildirir.
​Sonuçların Gösterimi: Hesaplanan kökler iomanip kütüphanesi kullanılarak virgülden sonra iki basamak olacak şekilde (precision) ekrana yansıtılır.
​Bu çalışma temel programlama mantığı ile matematiksel teoriyi birleştiren bir uygulamadır.
